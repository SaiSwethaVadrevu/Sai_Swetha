# Stock Price Forecasting using LSTM and Azure Machine Learning

## Overview

This project demonstrates an end-to-end data engineering and machine learning workflow for forecasting stock prices using historical market data. The solution focuses on collecting, preprocessing, transforming, and analyzing time-series data before training an LSTM (Long Short-Term Memory) model for stock price prediction. Azure Machine Learning was used to support model experimentation, while Python was utilized for data processing, feature engineering, model training, and evaluation.
